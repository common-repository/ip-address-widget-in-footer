This plugin allows you to show visitor's IP address in the footer section of the theme.

== Description ==

This plugin allows you to show visitor public IP Address in the footer section. This plugin calls [ipaddress.tech widget] (https://ipaddress.tech/widgets) which is basically an image (what-is-my-ip-address.png). When browser access this image URL, It detects its public IP Address and generate the image dynamically. You can change widget image background from admin setting area.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ip-address-widget-footer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->IPaddress.Tech Widget for more settings.

== Frequently Asked Questions ==

= How to install the plugin? =

Download the .zip file from [plugin repository](https://downloads.wordpress.org/plugin/ip-address-widget-in-footer.zip) website and Upload the plugin into your WordPress website. Then, Install the plugin and activate the plugin. You can also install the plugin through the WordPress plugins screen directly.

= How can i change widget theme =

Go through the IPAddress.tech Widget page under settings, and you can select widget theme and submit it.
